ShowBucket

ShowBucket is a web application for horse show management. It provides an online platform for show organizers to create and manage horse shows. It also allows horse show competitors to register for shows online, view show results, and search for nearby/upcoming shows.

Laravel, MySQL, HTML, CSS, Bootstrap, PHP
